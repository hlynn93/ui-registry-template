## Installation

Install any AUI component using the `shadcn` CLI:

```bash
npx shadcn@latest add https://acme.com/r/<component-name>.json
```

### Available components

| Component | Command |
| --- | --- |
| Hello World | `npx shadcn@latest add https://acme.com/r/hello-world.json` |
| Example Form | `npx shadcn@latest add https://acme.com/r/example-form.json` |
| Feature Card | `npx shadcn@latest add https://acme.com/r/feature-card.json` |
| Complex Component | `npx shadcn@latest add https://acme.com/r/complex-component.json` |
| Example with CSS | `npx shadcn@latest add https://acme.com/r/example-with-css.json` |
